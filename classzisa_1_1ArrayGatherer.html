Detailed Description

template<class T, int n_dims>
class zisa::ArrayGatherer< T, n_dims >

Gather a distributed array.

The array is, distributed along the first axis, i.e. for all k & l x(i, k) i = 0, ..., 9 lives on rank 0 x(i, k) i = 10, ..., 17 lives on rank 1 etc.

Task: Gather the distributed array on a subset of the total ranks.
